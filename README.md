## Tools Used

- Python
- Pandas
- NumPy
- SQLite
- Power BI
- Git
- GitHub
  

## Project Overview

This project analyzes Airbnb listings to understand how seasonal tourism influences pricing, occupancy, availability, revenue, and customer behavior across major cities.

The project is designed as a reusable analytics pipeline. The current implementation uses New York City as the first completed case study, with the same workflow intended to be extended to additional cities such as Paris, London, Tokyo, and Barcelona.
